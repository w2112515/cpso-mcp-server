# 📁 CPSO-Protocol 目录结构说明

该目录包含了企业级递归多智能体编排框架的所有核心模块。

## 📂 模块说明

- **audit/** - 对应战略授权书中的 Adversarial Audit (对抗性审计) 节点
- **ingestion/** - 对应战略授权书中的 Ingestion Node (摄入节点)
- **intelligence/** - 对应战略授权书中的 Intelligence Fusion (情报融合) 节点
- **scouts/** - 对应战略授权书中的 Parallel Scouting (并行侦察) 节点
- **state/** - 系统状态管理模块
- **strategy/** - 对应战略授权书中的 Strategy Generation (战略生成) 节点
- **tech/** - 对应战略授权书中的 Technical Officer (技术官) 节点
- **utils/** - 工具类模块
